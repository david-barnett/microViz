# microViz: microbiome data analysis and visualization

microViz provides functions for statistics and visualization of
microbiome sequencing data. microViz wraps, extends and complements
popular microbial ecology packages like phyloseq, vegan, and microbiome.

Check out the website for tutorials and illustrated help pages.

<https://david-barnett.github.io/microViz/>
